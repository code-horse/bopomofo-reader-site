# Bopomofo Reader - 사용 가이드

> 버전: v1.5.1

## 소개

Bopomofo Reader는 중국어 학습자를 위해 설계된 브라우저 확장 프로그램입니다. 향상된 주음 엔진(현대 중국어 사전 및 200개 이상의 다음자 보정 포함)으로 웹 페이지의 한자에 정확한 주음(보모포) 발음 주석을 추가하여 중국어 발음 학습을 돕습니다.

---

## 주요 기능

- **텍스트 선택 주석** — 웹 페이지에서 중국어 텍스트를 선택하면 주음과 음성 버튼이 자동으로 표시
- **전체 페이지 주음 모드** — 한 번의 클릭으로 페이지의 모든 한자에 주음 주석 추가
- **음성 읽기** — 스피커 버튼을 클릭하여 표준 중국어 발음 듣기
- **선택 영역 읽기 + 가라오케 효과** — 중국어 텍스트를 선택하면 읽기 및 번역 버튼이 있는 컴팩트 도구 모음이 표시되고, 음성에 맞춰 단어 또는 글자가 실시간으로 강조 표시(가라오케 효과)
- **선택 영역 번역** — 텍스트를 선택하고 도구 모음의 번역 버튼을 클릭하여 Bing 또는 Google 번역으로 즉시 번역, 인라인 버블로 표시
- **호버 사전** — 주석이 있는 한자에 마우스를 올리면 주음, CC-CEDICT 영어 사전 정의(11만 항목, TOCFL 등급), 발음 버튼 표시
- **PDF 리더** — PDF 문서의 중국어 텍스트에 자동으로 주음 주석을 추가하는 내장 PDF 리더
- **다양한 주음 스타일** — 성조 표시(hàn yǔ) 및 성조 없음(han yu) 지원
- **키보드 단축키** — 사용자 지정 키보드 단축키로 핵심 기능에 빠르게 접근
- **다국어 인터페이스** — 38개 인터페이스 언어 지원

## 사용 방법

### 1단계: 확장 프로그램 설치

[Chrome 웹 스토어](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg)에서 **Bopomofo Reader**를 설치하거나 개발자 모드로 로컬에서 로드합니다.

### 2단계: 웹 페이지 열기

중국어 콘텐츠가 포함된 웹 페이지를 방문합니다.

### 3단계: 텍스트 선택 또는 플로팅 버튼 사용

주석을 추가할 중국어 텍스트를 선택하거나, 페이지 오른쪽 하단의 플로팅 버튼을 클릭하여 전체 페이지 주음 모드를 활성화합니다.

### 4단계: 주음 확인

한자에 마우스를 올려 주음 툴팁을 확인하고, 스피커 아이콘을 클릭하여 발음을 들습니다.

### 5단계: 선택 텍스트 읽기 및 번역

마우스로 중국어 텍스트를 선택하면 두 개의 버튼이 있는 컴팩트 도구 모음이 표시됩니다:
- **🔊 읽기** — 선택한 텍스트를 가라오케 스타일 강조와 함께 읽기
- **🌐 번역** — 도구 모음 아래에 번역 버블 표시

또는 우클릭하여 « Bopomofo Reader > 선택 영역 읽기 » 또는 « Bopomofo Reader > 선택 영역 번역 »을 선택할 수 있습니다.

> **팁:** 브라우저 도구 모음의 확장 프로그램 아이콘을 클릭하여 설정 패널을 열고 주음 스타일, 읽기 속도, 번역 엔진 등을 조정할 수 있습니다.

## 선택 영역 읽기 및 가라오케

선택 영역 읽기 기능으로 중국어 텍스트를 한 번의 클릭으로 읽을 수 있습니다 — 문장 발음 학습 및 읽기 연습에 적합합니다.

**방법 1: 선택 도구 모음**  
마우스로 중국어 텍스트를 선택합니다. 🔊 읽기 및 🌐 번역 버튼이 있는 컴팩트 도구 모음이 표시됩니다. 읽기를 클릭하여 재생. 단어 또는 글자가 실시간으로 강조 표시(가라오케 효과).

**방법 2: 우클릭 메뉴**  
중국어 텍스트를 선택하고 우클릭하여 « Bopomofo Reader > 선택 영역 읽기 »를 선택합니다.

**방법 3: 키보드 단축키**  
텍스트를 선택하고 `Alt+Shift+S`(Mac: `Ctrl+Shift+S`)를 누릅니다.

> **팁:** 가라오케 강조 효과는 브라우저가 TTS 단어 경계 이벤트를 지원할 때 가장 효과적입니다. 지원하지 않으면 확장 프로그램이 시간 기반 대체 방법을 사용합니다.

## 번역

페이지의 텍스트를 선택하고 번역 기능을 사용하여 즉시 번역을 받을 수 있습니다.

**방법 1: 선택 도구 모음**  
텍스트를 선택한 후 🌐 번역 버튼을 클릭합니다. 결과와 복사 버튼이 있는 번역 버블이 표시됩니다.

**방법 2: 우클릭 메뉴**  
텍스트를 선택하고 우클릭하여 « Bopomofo Reader > 선택 영역 번역 »을 선택합니다.

**방법 3: 키보드 단축키**  
텍스트를 선택하고 `Alt+Shift+T`(Mac: `Ctrl+Shift+T`)를 누릅니다.

**번역 엔진:**
- **Bing 번역**(기본값) — Microsoft Translator 제공
- **Google 번역** — Google 제공

두 엔진 모두 **108개 대상 언어**를 지원합니다.

확장 프로그램 설정에서 번역 엔진과 대상 언어를 전환할 수 있습니다. 대상 언어는 브라우저 언어에서 자동으로 감지됩니다.

> **팁:** 도구 모음이나 버블 밖을 클릭하면 닫힙니다.

## PDF 리더

Bopomofo Reader에는 PDF 문서의 중국어 텍스트에 자동으로 주음 주석을 추가하는 내장 PDF 리더가 포함되어 있습니다.

### PDF 열기

- **자동 리다이렉트**: 브라우저에서 열린 PDF는 자동으로 Bopomofo Reader PDF 뷰어로 리다이렉트
- **드래그 앤 드롭**: 로컬 PDF 파일을 뷰어에 드롭
- **URL 붙여넣기**: PDF 링크를 뷰어에 직접 붙여넣기
- **팝업에서**: 확장 프로그램 아이콘을 클릭한 후 « PDF 리더 열기 » 클릭

### PDF 기능

| 기능 | 설명 |
|------|------|
| **사전 검색** | 한자를 클릭하여 즉시 영어 정의, 주음 읽기, TOCFL 등급 확인 |
| **사전 모드** | 한자를 클릭하여 영어 정의, 주음 발음, TOCFL 등급 배지 확인 |
| **클릭 사전** | 중국어 텍스트를 클릭하여 사전 정의(CC-CEDICT 영어 사전, 11만 항목, TOCFL 등급) 및 발음 확인 |
| **선택 도구 모음** | 텍스트를 선택한 후 도구 모음으로 읽기, 번역, 복사 |
| **텍스트 검색** | 중국어 텍스트로 검색 |
| **사이드바** | 목차, 페이지 썸네일, 탐색 |
| **3가지 테마** | 다크, 라이트, 세피아 읽기 모드 |
| **번역** | 텍스트를 선택하여 Bing 또는 Google 번역으로 번역 |

### PDF 키보드 단축키

| 키 | 동작 |
|----|------|
| ← / → | 이전 / 다음 페이지 |
| + / - | 확대 / 축소 |
| Ctrl+F / Cmd+F | 검색 열기 |
| Escape | 모든 플로팅 패널 닫기 |

> **참고:** 비밀번호로 보호된 PDF 및 스캔 PDF(이미지만)는 주음 주석을 위한 텍스트 레이어가 없어 지원되지 않습니다.

## 자주 묻는 질문

**Q: 일부 페이지에서 작동하지 않는 이유는?**  
A: 보안상의 이유로 `chrome://`, 브라우저 설정, Chrome 웹 스토어 등 특수 페이지에서는 확장 프로그램이 실행되지 않습니다.

**Q: 주음이 부정확한 경우?**  
A: 다음자의 주음에 오류가 있을 수 있습니다. 지속적으로 개선합니다. 구체적인 사례를 공유해 주세요.

**Q: 음성 읽기에 소리가 없음?**  
A: 시스템 볼륨 설정을 확인하고 중국어 음성 패키지가 설치되어 있는지 확인하세요. 브라우저와 OS에 따라 음성 지원이 다릅니다.

**Q: 전체 페이지 모드가 레이아웃에 영향?**  
A: 주음 주석에는 추가 공간이 필요하여 페이지 레이아웃에 영향을 줄 수 있습니다. 전체 페이지 모드를 비활성화하고 호버 툴팁을 사용하세요.

**Q: 번역이 작동하지 않음?**  
A: 번역에는 인터넷 연결이 필요합니다. Bing 번역이 실패하면 설정에서 Google 번역으로 전환하세요. 일부 네트워크 환경에서는 번역 서비스 접근이 차단될 수 있습니다.

## 오픈소스 사전 표기

Bopomofo Reader는 오픈소스 또는 커뮤니티 프로젝트의 오프라인 발음 및 사전 리소스를 포함합니다:

- **주음 변환**: pinyin-pro 및 @pinyin-pro/data/modern이 로컬 표준 중국어 발음 변환 및 단어 수준 보정을 제공; Bopomofo Reader는 피린 음절을 로컬에서 주음(Zhuyin/Bopomofo)으로 변환하고 대만 발음 우선 다음자 보정을 적용합니다.
- **중영 사전**: CC-CEDICT가 영어 정의 및 기본 사전 항목을 제공합니다.
- **다국어 정의**: CFDICT(프랑스어), HanDeDict(독일어), kaikki.org의 Wiktionary 파생 데이터가 일본어, 한국어, 베트남어, 중국어 정의를 제공합니다.
- **문자 변환**: OpenCC-js 및 번들 로컬 변환 모듈이 필요시 간체/번체 중국어 변환에 사용됩니다.

모든 사전 검색은 브라우저에서 로컬로 실행됩니다. 상위 프로젝트는 자체 저작권 및 라이선스를 유지합니다. 자세한 내용은 개인정보 보호정책을 참조하세요.
