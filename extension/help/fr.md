# Bopomofo Reader - Guide d'utilisation

> Version : v1.4.1

## Introduction

Bopomofo Reader est une extension de navigateur conçue pour les apprenants du chinois. Alimentée par un moteur bopomofo amélioré (avec dictionnaire de chinois moderne et plus de 200 corrections de caractères polyphones), elle ajoute avec précision des annotations de prononciation bopomofo aux caractères chinois sur les pages web.

---

## Fonctionnalités principales

- **Annotation par sélection** — Sélectionnez du texte chinois sur une page web pour afficher automatiquement le bopomofo et les boutons de prononciation
- **Mode bopomofo pleine page** — Ajoutez des annotations bopomofo à tous les caractères chinois de la page en un clic
- **Synthèse vocale** — Cliquez sur le bouton haut-parleur pour entendre la prononciation en mandarin standard
- **Lecture de la sélection** — Sélectionnez n'importe quel texte chinois et utilisez le bouton flottant ou le menu contextuel pour la lecture à voix haute
- **Info-bulles au survol** — Survolez les caractères annotés pour voir le bopomofo et les boutons de prononciation
- **Plusieurs styles de bopomofo** — Avec des marques de tons (hàn yǔ) ou sans tons (han yu)
- **Interface multilingue** — Prise en charge de 38 langues d'interface
- **Selection Speech with Karaoke Effect** — Select any Chinese text; a compact toolbar appears with speak and translate buttons; speech plays with real-time word-by-word highlighting (karaoke effect)
- **Selection Translation** — Select any text, click the translate button to get instant translation via Bing or Google Translate, displayed in an inline bubble
- **Hover Dictionary** — Hover over annotated characters to see Bopomofo, dictionary definitions from Taiwan MOE Revised Dictionary (155K+ entries), and pronunciation buttons
- **Keyboard Shortcuts** — Quick access via Alt+Shift+B (toggle), Alt+Shift+S (speak), Alt+Shift+T (translate)

## Comment utiliser

### Étape 1 : Installer l'extension

Installez **Bopomofo Reader** depuis le [Chrome Web Store](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg), ou chargez-la en mode développeur.

### Étape 2 : Ouvrir une page web

Visitez n'importe quelle page contenant du contenu chinois.

### Étape 3 : Sélectionner du texte ou utiliser le bouton flottant

Sélectionnez le texte chinois à annoter, ou cliquez sur le bouton flottant en bas à droite pour activer le mode bopomofo pleine page.

### Étape 4 : Voir le bopomofo

Survolez les caractères pour voir les info-bulles bopomofo, cliquez sur l'icône haut-parleur pour entendre la prononciation.

### Étape 5 : Lire le texte sélectionné

Sélectionnez du texte chinois avec la souris, cliquez sur le bouton flottant 🔊 pour lire ; ou faites un clic droit et choisissez « Lire la sélection ».

## Selection Speech & Karaoke

Select any Chinese text and read it aloud with one click — perfect for learning sentence pronunciation.

**Method 1: Selection Toolbar**
Select Chinese text with your mouse. A compact toolbar appears near the selection with a 🔊 speak button and a 🌐 translate button. Click speak to play. Words highlight in real time (karaoke effect).

**Method 2: Right-Click Menu**
Select text, right-click and choose "Bopomofo Reader > Speak Selection".

**Method 3: Keyboard Shortcut**
Select text and press `Alt+Shift+S` (Mac: `Ctrl+Shift+S`).

---

## Translation

Select any text on the page to get instant translations.

**Method 1: Selection Toolbar**
Select text, then click the 🌐 translate button. A translation bubble appears with the result and a copy button.

**Method 2: Right-Click Menu**
Select text, right-click and choose "Bopomofo Reader > Translate Selection".

**Method 3: Keyboard Shortcut**
Select text and press `Alt+Shift+T` (Mac: `Ctrl+Shift+T`).

**Translation Engines:**
- **Bing Translate** (default) — Powered by Microsoft Translator
- **Google Translate** — Powered by Google

Both engines support **108 target languages**.

## Built-in PDF Reader

Bopomofo Reader v1.4.1 includes a powerful built-in PDF reader that automatically adds Bopomofo annotations to any PDF file.

### Opening PDFs

- **Automatic**: Any PDF opened in the browser is automatically redirected to the Bopomofo PDF reader
- **From Popup**: Click the extension icon and choose "Open PDF Reader"
- **Drag & Drop**: Drag a local PDF file into the reader
- **Paste URL**: Paste any PDF link directly into the viewer

### PDF Features

| Feature | Description |
|---------|-------------|
| **Per-character Bopomofo** | Accurate Bopomofo annotation above each Chinese character, scaled with zoom |
| **4 Bopomofo Modes** | Tone marks, no tones, hover-only, or hidden |
| **Click Dictionary** | Click any Chinese character to see dictionary definitions (Taiwan MOE Revised Dictionary, 155K entries) |
| **Selection Toolbar** | Select text, then use the toolbar to read aloud, translate, or copy |
| **Sidebar** | Table of contents, page thumbnails, and Bopomofo search |
| **3 Themes** | Dark, Light, and Sepia reading modes |
| **Zoom Adaptive** | Bopomofo scales perfectly with zoom level, no overlap |
| **Keyboard Shortcuts** | ←→ page navigation, +- zoom, Ctrl/Cmd+F search, Escape to dismiss popups |

> **Tip:** The PDF reader works with both local files and online PDFs. All Bopomofo processing happens locally in your browser.

## Guide des paramètres

| Paramètre | Description |
|------------|-------------|
| **Activer le Bopomofo** | Interrupteur principal pour activer ou désactiver la fonction d'annotation bopomofo |
| **Bopomofo pleine page** | Affiche le bopomofo pour tous les caractères chinois (peut affecter la mise en page) |
| **Style de bopomofo** | Choisir entre marques de tons ou sans tons |
| **Vitesse de lecture** | Ajuster la vitesse de lecture des phrases (la prononciation des caractères individuels n'est pas affectée) |
| **Info-bulles au survol** | Afficher l'info-bulle bopomofo au survol de la souris |

## FAQ

**Q : Pourquoi ça ne fonctionne pas sur certaines pages ?**  
R : Pour des raisons de sécurité, les extensions ne peuvent pas fonctionner sur des pages spéciales comme `chrome://` ou les paramètres du navigateur.

**Q : Que faire si le bopomofo est incorrect ?**  
R : Le bopomofo des caractères polyphones peut comporter des erreurs. Nous améliorons continuellement. N'hésitez pas à nous signaler des cas spécifiques.

**Q : Pas de son pour la synthèse vocale ?**  
R : Vérifiez vos paramètres de volume et assurez-vous que les packs vocaux chinois sont installés.
