# Bopomofo Reader - 使用ガイド

> バージョン：v1.5.1

## はじめに

Bopomofo Readerは、中国語学習者向けのブラウザ拡張機能です。強化版注音エンジン（現代中国語辞書と200以上の多音字補正を搭載）により、ウェブページ上の漢字に正確な注音（ボポモフォ）注釈を追加し、中国語の発音学習をサポートします。

---

## 主な機能

- **テキスト選択注釈** — ウェブページ上の中国語テキストを選択すると、自動で注音と音声ボタンを表示
- **ページ全体注音モード** — ワンクリックでページ上のすべての漢字に注音を追加
- **音声読み上げ** — スピーカーボタンをクリックして標準中国語の発音を聞く
- **選択テキスト読み上げ＋カラオケ効果** — 任意の中国語テキストを選択すると、読み上げと翻訳ボタン付きのコンパクトなツールバーが表示され、音声に同期して単語や文字がリアルタイムでハイライト（カラオケ効果）
- **選択テキスト翻訳** — 任意のテキストを選択し、ツールバーの翻訳ボタンで Bing または Google 翻訳による即時翻訳をインラインバブルで表示
- **ホバー辞書** — 注釈付き漢字にマウスを合わせると注音、CC-CEDICT 英語辞書の定義（11万エントリ、TOCFL レベル）、発音ボタンを表示
- **PDFリーダー** — PDF文書内の中国語テキストに自動で注音注釈を追加する内蔵PDFリーダー
- **複数の注音スタイル** — 声調記号付き（hàn yǔ）と声調なし（han yu）の切り替え
- **キーボードショートカット** — カスタマイズ可能なショートカットで主要機能に素早くアクセス
- **多言語インターフェース** — 38言語のインターフェースに対応

## 使い方

### ステップ1：拡張機能をインストール

[Chrome ウェブストア](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg)から「Bopomofo Reader」をインストールするか、デベロッパーモードでローカルから読み込みます。

### ステップ2：任意のウェブページを開く

中国語コンテンツを含む任意のウェブページにアクセスします。

### ステップ3：テキストを選択するかフローティングボタンを使用

注釈したい中国語テキストを選択するか、ページ右下のフローティングボタンをクリックしてページ全体注音モードを有効にします。

### ステップ4：注音を確認

漢字にマウスを合わせて注音ツールチップを確認し、スピーカーアイコンをクリックして発音を聞きます。

### ステップ5：選択テキストの読み上げと翻訳

マウスで任意の中国語テキストを選択すると、コンパクトなツールバーが2つのボタン付きで表示されます：
- **🔊 読み上げ** — 選択テキストをカラオケ風ハイライト付きで読み上げ
- **🌐 翻訳** — ツールバー下に翻訳バブルを表示

右クリックから「Bopomofo Reader > 選択テキストを読み上げ」または「Bopomofo Reader > 選択テキストを翻訳」も選択できます。

> **ヒント：** ブラウザツールバーの拡張機能アイコンをクリックすると、設定パネルが開き、注音スタイル、読み上げ速度、翻訳エンジンなどを調整できます。

## 選択テキスト読み上げとカラオケ

選択テキスト読み上げ機能で、任意の中国語テキストをワンクリックで読み上げ — 文の発音学習や読解練習に最適です。

**方法1：選択ツールバー**  
マウスで中国語テキストを選択。🔊読み上げと🌐翻訳ボタン付きのコンパクトなツールバーが表示されます。読み上げをクリックして再生。単語や文字がリアルタイムでハイライト（カラオケ効果）。

**方法2：右クリックメニュー**  
中国語テキストを選択し、右クリックして「Bopomofo Reader > 選択テキストを読み上げ」を選択。

**方法3：キーボードショートカット**  
テキストを選択して `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）を押す。

> **ヒント：** カラオケハイライトは、ブラウザがTTS単語境界イベントをサポートしている場合に最も効果的です。非対応時はタイミングベースのフォールバックを使用します。

## 翻訳

ページ上の任意のテキストを選択し、翻訳機能で即時翻訳を取得できます。

**方法1：選択ツールバー**  
テキストを選択し、🌐翻訳ボタンをクリック。翻訳バブルが結果とコピーボタン付きで表示されます。

**方法2：右クリックメニュー**  
テキストを選択し、右クリックして「Bopomofo Reader > 選択テキストを翻訳」を選択。

**方法3：キーボードショートカット**  
テキストを選択して `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）を押す。

**翻訳エンジン：**
- **Bing 翻訳**（デフォルト）— Microsoft Translator 提供
- **Google 翻訳** — Google 提供

両エンジンとも **108言語** の翻訳先をサポート。

拡張機能の設定で翻訳エンジンと翻訳先言語を切り替え可能。翻訳先言語はブラウザの言語から自動検出されます。

> **ヒント：** ツールバーやバブルの外をクリックすると閉じます。

## PDFリーダー

Bopomofo Readerには、PDF文書内の中国語テキストに自動で注音注釈を追加する内蔵PDFリーダーが含まれています。

### PDFを開く

- **自動リダイレクト**：ブラウザで開いたPDFは自動的にBopomofo Reader PDFビューアにリダイレクト
- **ドラッグ＆ドロップ**：ローカルPDFファイルをビューアにドロップ
- **URL貼り付け**：PDFリンクをビューアに直接貼り付け
- **ポップアップから**：拡張機能アイコンをクリックし、「PDFリーダーを開く」をクリック

### PDF機能

| 機能 | 説明 |
|------|------|
| **辞書検索** | 漢字をクリックして即座に英語定義、注音読み、TOCFLレベルを表示 |
| **辞書モード** | 漢字をクリックして英語定義、注音発音、TOCFLレベルバッジを表示 |
| **クリック辞書** | 中国語テキストをクリックして辞書定義（CC-CEDICT英語辞書、11万エントリ、TOCFLレベル）と発音を表示 |
| **選択ツールバー** | テキストを選択し、ツールバーで読み上げ、翻訳、コピー |
| **テキスト検索** | 中国語テキストで検索 |
| **サイドバー** | 目次、ページサムネイル、ナビゲーション |
| **3テーマ** | ダーク、ライト、セピアの読書モード |
| **翻訳** | テキストを選択してBingまたはGoogle翻訳で翻訳 |

### PDFキーボードショートカット

| キー | 操作 |
|------|------|
| ← / → | 前のページ / 次のページ |
| + / - | ズームイン / ズームアウト |
| Ctrl+F / Cmd+F | 検索を開く |
| Escape | すべてのフローティングパネルを閉じる |

> **注意：** パスワード保護されたPDFやスキャンPDF（画像のみ）は、注音注釈用のテキストレイヤーがないためサポートされていません。

## 設定ガイド

| 設定 | 説明 |
|------|------|
| **注音を有効化** | 注音注釈機能のマスタースイッチ |
| **ページ全体注音** | すべての漢字に注音を表示（ページレイアウトに影響する可能性あり） |
| **注音スタイル** | 声調記号付きまたは声調なしの表示を選択 |
| **文の読み上げ速度** | 選択テキスト読み上げの速度を調整（単字発音は影響なし） |
| **ホバーモード** | ホバー動作：辞書（CC-CEDICT英語辞書とTOCFLレベル）、注音のみ、またはオフ |
| **翻訳エンジン** | Bing翻訳またはGoogle翻訳を選択 |
| **翻訳先言語** | 翻訳先言語を設定（ブラウザ言語から自動検出） |
| **PDF検出** | PDFファイルを内蔵Bopomofo Reader PDFビューアに自動リダイレクト |

## よくある質問

**Q：一部のページで動作しないのはなぜ？**  
A：セキュリティ上、`chrome://`、ブラウザ設定、Chromeウェブストアなどの特殊ページでは拡張機能は動作しません。

**Q：注音が不正確な場合は？**  
A：多音字の注音には誤りがある場合があります。継続的に改善しています。具体的な事例をお知らせください。

**Q：音声読み上げに音が出ない？**  
A：システムの音量設定を確認し、中国語音声パックがインストールされていることを確認してください。ブラウザとOSによって音声サポートは異なります。

**Q：ページ全体モードがレイアウトに影響する？**  
A：注音注釈には追加スペースが必要で、ページレイアウトに影響する可能性があります。ページ全体モードを無効にしてホバーツールチップを使用してください。

**Q：翻訳が動作しない？**  
A：翻訳にはインターネット接続が必要です。Bing翻訳が失敗した場合、設定でGoogle翻訳に切り替えてください。一部のネットワーク環境では翻訳サービスへのアクセスがブロックされる場合があります。

## オープンソース辞書の帰属

Bopomofo Readerは、オープンソースまたはコミュニティプロジェクトのオフライン発音・辞書リソースを同梱しています：

- **注音変換**：pinyin-proと@pinyin-pro/data/modernがローカルでの標準中国語発音変換と語レベルの補正を提供；Bopomofo Readerはピンイン音節をローカルで注音（Zhuyin/Bopomofo）に変換し、台湾発音優先の多音字補正を適用。
- **中英辞書**：CC-CEDICTが英語定義と基本辞書エントリを提供。
- **多言語定義**：CFDICT（フランス語）、HanDeDict（ドイツ語）、kaikki.orgのWiktionary派生データが日本語、韓国語、ベトナム語、中国語の定義を提供。
- **文字変換**：OpenCC-jsと同梱のローカル変換モジュールが必要に応じて簡体字/繁体字変換に使用。

すべての辞書検索はブラウザ内でローカルに実行されます。上流プロジェクトは独自の著作権とライセンスを保持；詳細はプライバシーポリシーを参照。
