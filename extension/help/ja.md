# Bopomofo Reader - 使用ガイド

> バージョン：v1.3.0

## はじめに

Bopomofo Readerは、中国語学習者向けのブラウザ拡張機能です。強化版ピンインエンジン（現代中国語辞書と200以上の多音字補正を搭載）を採用し、ウェブページ上の漢字にピンイン（発音記号）を正確に注釈として追加し、中国語の発音学習をサポートします。

---

## 主な機能

- **テキスト選択注釈** — ウェブページ上の中国語テキストを選択すると、自動でピンインと音声ボタンを表示
- **ページ全体ピンインモード** — ワンクリックでページ上のすべての漢字にピンインを追加
- **音声読み上げ** — スピーカーボタンをクリックして標準中国語の発音を聞く
- **選択テキスト読み上げ** — 任意の中国語テキストを選択し、フローティングボタンまたは右クリックメニューから音声で読み上げ
- **ホバーツールチップ** — 注釈付き漢字にマウスを合わせるとピンインと発音ボタンを表示
- **複数のピンインスタイル** — 声調記号付き（hàn yǔ）と声調なし（han yu）の切り替え
- **多言語インターフェース** — 38言語のインターフェースに対応

## 使い方

### ステップ1：拡張機能をインストール

[Chrome ウェブストア](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg)から「Bopomofo Reader」をインストールするか、デベロッパーモードでローカルから読み込みます。

### ステップ2：任意のウェブページを開く

中国語コンテンツを含む任意のウェブページにアクセスします。

### ステップ3：テキストを選択するかフローティングボタンを使用

注釈したい中国語テキストを選択するか、ページ右下のフローティングボタンをクリックしてページ全体ピンインモードを有効にします。

### ステップ4：ピンインを確認

漢字にマウスを合わせてピンインツールチップを確認し、スピーカーアイコンをクリックして発音を聞きます。

### ステップ5：選択テキストを読み上げ

マウスで任意の中国語テキストを選択し、表示される🔊フローティングボタンをクリックして読み上げ。または右クリックから「選択テキストを読み上げ」を選択します。

> **ヒント：** ブラウザツールバーの拡張機能アイコンをクリックすると、設定パネルが開き、ピンインスタイルや読み上げ速度などを調整できます。

## 設定ガイド

| 設定項目 | 説明 |
|----------|------|
| **ピンインを有効化** | ピンイン注釈機能のマスタースイッチ |
| **ページ全体ピンイン** | 有効にするとすべての漢字にピンインを表示（ページレイアウトに影響する可能性あり） |
| **ピンインスタイル** | 声調記号付きまたは声調なし表示の選択 |
| **文章読み上げ速度** | 文章読み上げの速度を調整（単字発音には影響しません） |
| **ホバーツールチップ** | マウスホバー時にピンインツールチップを表示 |

## よくある質問

**Q：一部のページで機能しないのはなぜですか？**  
A：セキュリティ上の理由から、ブラウザ拡張機能は`chrome://`やブラウザ設定ページなどの特殊なページでは動作しません。

**Q：ピンインが不正確な場合は？**  
A：多音字のピンインには誤りがある場合があります。継続的に改善しておりますので、具体的なケースをフィードバックいただければ幸いです。

**Q：音声読み上げから音が出ない場合は？**  
A：システムの音量設定を確認し、中国語の音声パックがインストールされているか確認してください。音声サポートはブラウザやオペレーティングシステムによって異なります。
