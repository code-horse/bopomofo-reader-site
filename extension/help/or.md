# Bopomofo Reader - ଉପଯୋଗକର୍ତ୍ତା ଗାଇଡ୍

> ସଂସ୍କରଣ: v1.5.1

## ପରିଚୟ

Bopomofo Reader ଚାଇନି ଶିଖୁଥିବା ଲୋକଙ୍କ ପାଇଁ ବ୍ରାଉଜର ଏକ୍ସଟେନ୍ସନ୍। bopomofo, CC-CEDICT, ଅନୁବାଦ, କାରାଓକେ, PDF ରିଡର୍।

## ମୁଖ୍ୟ ବିଶେଷତାଗୁଡ଼ିକ

ପାଠ ଚୟନ, ପୂର୍ଣ୍ଣ ପୃଷ୍ଠା, TTS, କାରାଓକେ, ଅନୁବାଦ, CC-CEDICT, PDF, 38 ଭାଷା।

## ବ୍ୟବହାର

[Chrome Web Store](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg) ରୁ ଇନଷ୍ଟଲ୍ କରନ୍ତୁ। 🔊 ପଢ଼ନ୍ତୁ 🌐 ଅନୁବାଦ।

## କାରାଓକେ ପଠନ

`Alt+Shift+S` (Mac: `Ctrl+Shift+S`). TTS କାରାଓକେ ହାଇଲାଇଟ୍।

## ଅନୁବାଦ

`Alt+Shift+T` (Mac: `Ctrl+Shift+T`). Bing, Google. 108 ଭାଷା।

## PDF ରିଡର୍

ଅଟୋ-ରିଡାଇରେକ୍ଟ, ଶବ୍ଦକୋଷ, 3 ଥିମ୍। ପାସୱର୍ଡ PDF ସମର୍ଥିତ ନୁହେଁ।

## ସର୍ଟକଟ୍

`Alt+Shift+B`/`S`/`T`. `chrome://extensions/shortcuts`。
