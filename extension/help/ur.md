# Bopomofo Reader - صارف گائیڈ

> ورژن: v1.5.1

## تعارف

Bopomofo Reader چینی سیکھنے والوں کے لیے براؤزر ایکسٹینشن۔ bopomofo، CC-CEDICT، ترجمہ، کراوکی، PDF ریڈر۔

## اہم خصوصیات

ٹیکسٹ سلیکشن، پورا صفحہ، TTS، کراوکی، ترجمہ، hover ڈکشنری، PDF، bopomofo سٹائلز، شارٹ کٹس، 38 زبانیں۔

## استعمال

[Chrome Web Store](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg) سے انسٹال کریں۔

## کراوکی

`Alt+Shift+S` (Mac: `Ctrl+Shift+S`).

## ترجمہ

`Alt+Shift+T` (Mac: `Ctrl+Shift+T`). Bing اور Google۔

## PDF ریڈر

خودکار ری ڈائریکٹ، ڈکشنری، 3 تھیمز۔

## کی بورڈ شارٹ کٹس

Alt+Shift+B/S/T.
