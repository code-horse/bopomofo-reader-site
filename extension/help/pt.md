# Bopomofo Reader - Guia do Usuário

> Versão: v1.3.0

## Introdução

Bopomofo Reader é uma extensão de navegador projetada para estudantes de chinês. Com um mecanismo de bopomofo aprimorado (com dicionário de chinês moderno e mais de 200 correções de caracteres polifônicos), adiciona anotações de pronúncia bopomofo aos caracteres chineses em páginas da web com precisão.

---

## Funcionalidades principais

- **Anotação por seleção** — Selecione texto chinês em páginas da web para exibir automaticamente o bopomofo e botões de pronúncia
- **Modo bopomofo de página inteira** — Adicione anotações bopomofo a todos os caracteres chineses da página com um clique
- **Texto para fala** — Clique no botão do alto-falante para ouvir a pronúncia padrão do mandarim
- **Leitura de seleção** — Selecione qualquer texto chinês e use o botão flutuante ou o menu de contexto para ler em voz alta
- **Dicas ao passar o mouse** — Passe o mouse sobre caracteres anotados para ver o bopomofo e botões de pronúncia
- **Vários estilos de bopomofo** — Com marcas de tom (hàn yǔ) e sem tons (han yu)
- **Interface multilíngue** — Suporta 38 idiomas de interface

---

## Como usar

### Passo 1: Instalar a extensão

Instale o **Bopomofo Reader** na [Chrome Web Store](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg) ou carregue-o no modo desenvolvedor.

### Passo 2: Abrir qualquer página

Visite qualquer página com conteúdo em chinês.

### Passo 3: Selecionar texto ou usar botão flutuante

Selecione o texto chinês que deseja anotar, ou clique no botão flutuante no canto inferior direito para ativar o modo bopomofo de página inteira.

### Passo 4: Ver o bopomofo

Passe o mouse sobre os caracteres para ver as dicas de bopomofo, clique no ícone do alto-falante para ouvir a pronúncia.

### Passo 5: Ler texto selecionado

Selecione texto chinês com o mouse, clique no botão flutuante 🔊 para ler; ou clique com o botão direito e escolha "Ler seleção".

## Configurações

| Configuração | Descrição |
|--------------|-----------|
| **Ativar Bopomofo** | Interruptor principal para ativar/desativar a função de anotação bopomofo |
| **Bopomofo de página inteira** | Exibe bopomofo para todos os caracteres (pode afetar o layout da página) |
| **Estilo de bopomofo** | Escolha entre marcas de tom ou sem tons |
| **Velocidade de leitura** | Ajustar a velocidade de leitura de frases (a pronúncia de caracteres individuais não é afetada) |
| **Dicas ao passar o mouse** | Mostrar dica de bopomofo ao passar o mouse |

## Perguntas frequentes

**P: Por que não funciona em algumas páginas?**  
R: Por razões de segurança, extensões de navegador não podem ser executadas em páginas especiais como `chrome://` ou configurações do navegador.

**P: E se o bopomofo estiver incorreto?**  
R: O bopomofo para caracteres com múltiplas pronúncias pode conter erros. Estamos melhorando continuamente. Por favor, compartilhe casos específicos para nos ajudar a melhorar.

**P: Sem som no texto para fala?**  
R: Verifique as configurações de volume do sistema e certifique-se de que os pacotes de voz em chinês estejam instalados.
