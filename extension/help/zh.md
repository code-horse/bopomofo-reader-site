# 注音助读 - 使用说明

> 版本：v1.2.0

## 简介

注音助读是一款浏览器扩展，专为中文学习者设计。它采用增强版注音引擎（含现代汉语词典和200多组多音字智能纠正），可以自动为网页上的汉字添加精准的注音注音，帮助您更轻松地学习中文发音。

---

## 主要功能

- **选中文字标注** — 选中网页上的中文文字，自动显示注音注音和语音朗读按钮
- **全网页注音模式** — 一键为页面上所有汉字添加注音标注，方便阅读学习
- **语音朗读** — 点击发音按钮，听取汉字的标准普通话发音
- **选区朗读 + 卡拉OK效果** — 选中任意中文文本，弹出紧凑工具栏（朗读 + 翻译按钮），朗读时逐词或逐字高亮同步显示（卡拉OK效果）
- **选区翻译** — 选中文本后点击工具栏翻译按钮，通过 Bing 或 Google 翻译即时获取翻译结果，以悬浮气泡显示
- **悬停提示** — 鼠标悬停在已标注的汉字上，显示注音和发音按钮
- **多种注音样式** — 支持带声调符号（hàn yǔ）和不带声调（han yu）两种显示方式
- **快捷键** — 通过可自定义的键盘快捷键快速使用核心功能
- **多语言界面** — 支持 38 种界面语言切换

## 使用方法

### 第一步：安装扩展

在[谷歌应用商店](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg)安装「注音助读」，或通过开发者模式加载本地扩展。

### 第二步：打开任意网页

访问任何包含中文内容的网页。

### 第三步：选中文字或使用悬浮按钮

选中想要标注的中文文字，或点击页面右下角的悬浮按钮进入全网页注音模式。

### 第四步：查看注音

悬停在汉字上查看注音提示，点击扬声器图标收听发音。

### 第五步：朗读与翻译选中文本

用鼠标选中任意中文文本，选区附近会出现一个紧凑工具栏，包含两个按钮：
- **🔊 朗读** — 朗读选中文本，同时逐词或逐字高亮显示（卡拉OK效果）
- **🌐 翻译** — 在工具栏下方显示翻译结果气泡

也可以右键选择「注音助读 > 朗读选区」或「注音助读 > 翻译选区」。

> **提示：** 点击浏览器工具栏中的扩展图标，可以打开设置面板，调整注音样式、语速、翻译引擎等参数。

## 选区朗读与卡拉OK

选区朗读功能让您可以选中任意中文文本，一键朗读整段文字，非常适合学习句子发音和跟读练习。

**方式一：选区工具栏**  
用鼠标选中中文文本后，选区附近会出现一个紧凑工具栏，包含🔊朗读按钮和🌐翻译按钮。点击朗读按钮播放，播放时每个词或字会实时高亮显示（卡拉OK效果），帮助您跟读。

**方式二：右键菜单**  
选中中文文本后，右键选择「注音助读 > 朗读选区」。

**方式三：快捷键**  
选中文本后按 `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）即可朗读。

> **提示：** 卡拉OK高亮效果在浏览器支持 TTS 词边界事件时效果最佳。若不支持，扩展会使用基于计时的备用方案实现平滑高亮。

## 翻译

选中页面上的任意文本，使用翻译功能即可获取即时翻译。

**方式一：选区工具栏**  
选中文本后，点击工具栏中的🌐翻译按钮，下方会弹出翻译结果气泡，带有复制按钮。

**方式二：右键菜单**  
选中文本后，右键选择「注音助读 > 翻译选区」。

**方式三：快捷键**  
选中文本后按 `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）即可翻译。

**翻译引擎：**
- **Bing 翻译**（默认）— 由微软翻译提供支持
- **Google 翻译** — 由 Google 提供支持

两种引擎均支持 **108 种目标语言**。

您可以在扩展设置中切换翻译引擎和目标语言。目标语言会根据您的浏览器语言自动检测。

> **提示：** 点击工具栏或气泡外的任意位置即可关闭。

## 快捷键

| 快捷键 | Mac 快捷键 | 功能 |
|--------|-----------|------|
| `Alt+Shift+B` | `Ctrl+Shift+B` | 切换注音标注开关 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 朗读选中文本 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 翻译选中文本 |

> **提示：** 您可以在 Chrome 的 `chrome://extensions/shortcuts` 页面自定义这些快捷键。

## 设置说明

| 设置项 | 说明 |
|--------|------|
| **启用注音** | 总开关，控制是否启用注音标注功能 |
| **全网页注音** | 开启后为页面上所有汉字显示注音（可能影响页面布局） |
| **注音样式** | 选择带声调符号或不带声调的注音显示方式 |
| **句子朗读语速** | 调整选区朗读的速度（单字发音不受影响） |
| **悬停提示** | 鼠标悬停时显示注音提示框 |
| **翻译引擎** | 选择 Bing 翻译或 Google 翻译 |
| **目标语言** | 设置翻译的目标语言（根据浏览器语言自动检测） |

## 常见问题

**问：为什么某些页面无法使用？**  
答：出于安全原因，浏览器扩展无法在某些特殊页面（如`chrome://`、浏览器设置页面等）上运行。

**问：注音标注不准确怎么办？**  
答：多音字的注音可能存在误差，我们会持续优化。欢迎反馈具体案例帮助我们改进。

**问：语音朗读没有声音？**  
答：请检查系统音量设置，确保已安装中文语音包。不同浏览器和操作系统的语音支持可能有所不同。

**问：全网页注音模式影响了页面布局？**  
答：注音标注需要额外空间显示，可能会影响原有页面布局。如果影响阅读，建议关闭全网页模式，改用悬停查看。

**问：翻译不工作？**  
答：翻译功能需要网络连接。如果 Bing 翻译失败，请尝试在设置中切换为 Google 翻译。某些网络环境可能会阻止访问翻译服务。
