# 注音助讀 - 使用說明

> 版本：v1.4.0

## 簡介

注音助讀是一款瀏覽器擴充功能，專為中文學習者設計。它採用增強版注音引擎（含現代漢語詞典和200多組多音字智慧糾正），可以自動為網頁上的漢字添加精準的注音標注，幫助您更輕鬆地學習中文發音。

---

## 主要功能

- **選取文字標注** — 選取網頁上的中文文字，自動顯示注音和語音朗讀按鈕
- **全網頁注音模式** — 一鍵為頁面上所有漢字添加注音標注，方便閱讀學習
- **語音朗讀** — 點擊發音按鈕，聆聽漢字的標準普通話發音
- **選區朗讀 + 卡拉OK效果** — 選取任意中文文本，彈出緊湊工具列（朗讀 + 翻譯按鈕），朗讀時逐詞或逐字高亮同步顯示（卡拉OK效果）
- **選區翻譯** — 選取文本後點擊工具列翻譯按鈕，透過 Bing 或 Google 翻譯即時取得翻譯結果，以懸浮氣泡顯示
- **懸浮詞典** — 滑鼠懸停在已標注的漢字上，顯示注音、教育部《重編國語辭典修訂本》詞典釋義（15.5 萬詞條）和發音按鈕
- **PDF 閱讀器** — 內建 PDF 閱讀器，可自動為 PDF 文件中的中文漢字添加注音標注
- **多種注音樣式** — 支援帶聲調符號（hàn yǔ）和不帶聲調（han yu）兩種顯示方式
- **快速鍵** — 透過可自訂的鍵盤快速鍵快速使用核心功能
- **多語言介面** — 支援 38 種介面語言切換

## 使用方法

### 第一步：安裝擴充功能

在[Chrome 線上應用程式商店](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg)安裝「注音助讀」，或透過開發人員模式載入本地擴充功能。

### 第二步：開啟任意網頁

造訪任何包含中文內容的網頁。

### 第三步：選取文字或使用懸浮按鈕

選取想要標注的中文文字，或點擊頁面右下角的懸浮按鈕進入全網頁注音模式。

### 第四步：查看注音

懸停在漢字上查看注音提示，點擊揚聲器圖示收聽發音。

### 第五步：朗讀與翻譯選取文本

用滑鼠選取任意中文文本，選區附近會出現一個緊湊工具列，包含兩個按鈕：
- **🔊 朗讀** — 朗讀選取文本，同時逐詞或逐字高亮顯示（卡拉OK效果）
- **🌐 翻譯** — 在工具列下方顯示翻譯結果氣泡

也可以右鍵選擇「注音助讀 > 朗讀選區」或「注音助讀 > 翻譯選區」。

> **提示：** 點擊瀏覽器工具列中的擴充功能圖示，可以開啟設定面板，調整注音樣式、語速、翻譯引擎等參數。

## 選區朗讀與卡拉OK

選區朗讀功能讓您可以選取任意中文文本，一鍵朗讀整段文字，非常適合學習句子發音和跟讀練習。

**方式一：選區工具列**  
用滑鼠選取中文文本後，選區附近會出現一個緊湊工具列，包含🔊朗讀按鈕和🌐翻譯按鈕。點擊朗讀按鈕播放，播放時每個詞或字會即時高亮顯示（卡拉OK效果），幫助您跟讀。

**方式二：右鍵選單**  
選取中文文本後，右鍵選擇「注音助讀 > 朗讀選區」。

**方式三：快速鍵**  
選取文本後按 `Alt+Shift+S`（Mac：`Ctrl+Shift+S`）即可朗讀。

> **提示：** 卡拉OK高亮效果在瀏覽器支援 TTS 詞邊界事件時效果最佳。若不支援，擴充功能會使用基於計時的備用方案實現平滑高亮。

## 翻譯

選取頁面上的任意文本，使用翻譯功能即可取得即時翻譯。

**方式一：選區工具列**  
選取文本後，點擊工具列中的🌐翻譯按鈕，下方會彈出翻譯結果氣泡，帶有複製按鈕。

**方式二：右鍵選單**  
選取文本後，右鍵選擇「注音助讀 > 翻譯選區」。

**方式三：快速鍵**  
選取文本後按 `Alt+Shift+T`（Mac：`Ctrl+Shift+T`）即可翻譯。

**翻譯引擎：**
- **Bing 翻譯**（預設）— 由微軟翻譯提供支援
- **Google 翻譯** — 由 Google 提供支援

兩種引擎均支援 **108 種目標語言**。

您可以在擴充功能設定中切換翻譯引擎和目標語言。目標語言會根據您的瀏覽器語言自動偵測。

> **提示：** 點擊工具列或氣泡外的任意位置即可關閉。

## PDF 閱讀器

注音助讀內建 PDF 閱讀器，可自動為 PDF 文件中的中文漢字添加注音標注。

### 開啟 PDF

- **自動攔截**：在瀏覽器中開啟的 PDF 會自動跳轉到注音閱讀器
- **拖曳開啟**：將本地 PDF 檔案拖曳到閱讀器
- **貼上連結**：直接貼上 PDF 連結開啟
- **從彈出視窗**：點擊擴充功能圖示，然後點擊「開啟 PDF 閱讀器」

### PDF 功能

| 功能 | 說明 |
|------|------|
| **逐字注音** | 每個漢字上方精準標注注音，隨縮放自適應 |
| **4 種注音模式** | 聲調 / 無聲調 / 僅懸浮 / 隱藏 |
| **點擊查詞典** | 點擊漢字查看詞典釋義（教育部《重編國語辭典修訂本》15.5 萬詞條）和發音 |
| **選區工具列** | 選取文本後彈出工具列：朗讀、翻譯、複製 |
| **注音搜尋** | 支援中文和注音搜尋 |
| **側邊欄** | 目錄大綱、頁面縮圖、快速導覽 |
| **3 套佈景主題** | 暗色、淺色、護眼模式 |
| **翻譯** | 選取文本透過必應或 Google 翻譯 |

### PDF 快速鍵

| 按鍵 | 功能 |
|------|------|
| ← / → | 上一頁 / 下一頁 |
| + / - | 放大 / 縮小 |
| Ctrl+F / Cmd+F | 開啟搜尋 |
| Escape | 關閉所有浮動面板 |

> **注意：** 不支援加密 PDF 和掃描件（純圖片 PDF），因為它們沒有文字層可供標注注音。

## 快速鍵

| 快速鍵 | Mac 快速鍵 | 功能 |
|--------|-----------|------|
| `Alt+Shift+B` | `Ctrl+Shift+B` | 切換注音標注開關 |
| `Alt+Shift+S` | `Ctrl+Shift+S` | 朗讀選取文本 |
| `Alt+Shift+T` | `Ctrl+Shift+T` | 翻譯選取文本 |

> **提示：** 您可以在 Chrome 的 `chrome://extensions/shortcuts` 頁面自訂這些快速鍵。

## 設定說明

| 設定項 | 說明 |
|--------|------|
| **啟用注音** | 總開關，控制是否啟用注音標注功能 |
| **全網頁注音** | 開啟後為頁面上所有漢字顯示注音（可能影響頁面版面） |
| **注音樣式** | 選擇帶聲調符號或不帶聲調的注音顯示方式 |
| **句子朗讀語速** | 調整選區朗讀的速度（單字發音不受影響） |
| **懸浮模式** | 選擇懸浮行為：詞典模式（顯示教育部辭典釋義）、僅注音、或關閉 |
| **翻譯引擎** | 選擇 Bing 翻譯或 Google 翻譯 |
| **目標語言** | 設定翻譯的目標語言（根據瀏覽器語言自動偵測） |
| **PDF 偵測** | 自動偵測並重新導向 PDF 檔案到內建注音 PDF 閱讀器 |

## 常見問題

**問：為什麼某些頁面無法使用？**  
答：出於安全原因，瀏覽器擴充功能無法在某些特殊頁面（如`chrome://`、瀏覽器設定頁面等）上執行。

**問：注音標注不準確怎麼辦？**  
答：多音字的注音可能存在誤差，我們會持續優化。歡迎回饋具體案例幫助我們改進。

**問：語音朗讀沒有聲音？**  
答：請檢查系統音量設定，確保已安裝中文語音套件。不同瀏覽器和作業系統的語音支援可能有所不同。

**問：全網頁注音模式影響了頁面版面？**  
答：注音標注需要額外空間顯示，可能會影響原有頁面版面。如果影響閱讀，建議關閉全網頁模式，改用懸停查看。

**問：翻譯不工作？**  
答：翻譯功能需要網路連線。如果 Bing 翻譯失敗，請嘗試在設定中切換為 Google 翻譯。某些網路環境可能會阻止存取翻譯服務。
