# 注音助讀 - 使用說明

> 版本：v1.2.0

## 簡介

注音助讀是一款瀏覽器擴充功能，專為中文學習者設計。它採用增強版注音引擎（含現代漢語詞典和200多組多音字智慧糾正），可以自動為網頁上的漢字添加精準的注音注音，幫助您更輕鬆地學習中文發音。

---

## 主要功能

- **選取文字標注** — 選取網頁上的中文文字，自動顯示注音注音和語音朗讀按鈕
- **全網頁注音模式** — 一鍵為頁面上所有漢字添加注音標注，方便閱讀學習
- **語音朗讀** — 點擊發音按鈕，聆聽漢字的標準普通話發音
- **選區朗讀** — 選取任意中文文本，點擊浮動按鈕或右鍵選擇「朗讀選取文字」即可朗讀整段文字
- **懸停提示** — 滑鼠懸停在已標注的漢字上，顯示注音和發音按鈕
- **多種注音樣式** — 支援帶聲調符號（hàn yǔ）和不帶聲調（han yu）兩種顯示方式
- **多語言介面** — 支援 38 種介面語言切換

## 使用方法

### 第一步：安裝擴充功能

在[Chrome 線上應用程式商店](https://chromewebstore.google.com/detail/bopomofo-reader/jkibgddokcnhedkhmleaohojhlnemmkg)安裝「注音助讀」，或透過開發人員模式載入本地擴充功能。

### 第二步：開啟任意網頁

造訪任何包含中文內容的網頁。

### 第三步：選取文字或使用懸浮按鈕

選取想要標注的中文文字，或點擊頁面右下角的懸浮按鈕進入全網頁注音模式。

### 第四步：查看注音

懸停在漢字上查看注音提示，點擊揚聲器圖示收聽發音。

### 第五步：朗讀選取文本

用滑鼠選取任意中文文本，點擊出現的🔊浮動按鈕朗讀；或右鍵選擇「朗讀選取文字」。

> **提示：** 點擊瀏覽器工具列中的擴充功能圖示，可以開啟設定面板，調整注音樣式、語速等參數。

## 選區朗讀功能

選區朗讀功能讓您可以選取任意中文文本，一鍵朗讀整段文字，非常適合學習句子發音和跟讀練習。

**方式一：浮動按鈕**  
用滑鼠選取中文文本後，選區右上角會出現一個綠色的🔊按鈕，點擊即可朗讀。

**方式二：右鍵選單**  
選取中文文本後，點擊滑鼠右鍵，在選單中選擇「朗讀選取文字」。

> **提示：** 選區朗讀會使用您在設定中配置的句子語速。單字發音使用預設語速，不受設定影響。

## 設定說明

| 設定項 | 說明 |
|--------|------|
| **啟用注音** | 總開關，控制是否啟用注音標注功能 |
| **全網頁注音** | 開啟後為頁面上所有漢字顯示注音（可能影響頁面版面） |
| **注音樣式** | 選擇帶聲調符號或不帶聲調的注音顯示方式 |
| **句子朗讀語速** | 調整選區朗讀的速度（單字發音不受影響） |
| **懸停提示** | 滑鼠懸停時顯示注音提示框 |

## 常見問題

**問：為什麼某些頁面無法使用？**  
答：出於安全原因，瀏覽器擴充功能無法在某些特殊頁面（如`chrome://`、瀏覽器設定頁面等）上執行。

**問：注音標注不準確怎麼辦？**  
答：多音字的注音可能存在誤差，我們會持續優化。歡迎回饋具體案例幫助我們改進。

**問：語音朗讀沒有聲音？**  
答：請檢查系統音量設定，確保已安裝中文語音套件。不同瀏覽器和作業系統的語音支援可能有所不同。

**問：全網頁注音模式影響了頁面版面？**  
答：注音標注需要額外空間顯示，可能會影響原有頁面版面。如果影響閱讀，建議關閉全網頁模式，改用懸停查看。
